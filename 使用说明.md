# 如何将“智效待办”发布为手机 APP

恭喜！您的专属待办工具包已准备就绪。请按照以下步骤将其上传到 GitHub，即可生成分享链接并安装到手机。

## 第一步：上传到 GitHub

1.  **注册/登录 GitHub**：访问 [github.com](https://github.com/)。
2.  **创建新仓库**：
    *   点击右上角的 `+` 号 -> `New repository`。
    *   **Repository name** 填入 `todo-app`（或者任何你喜欢的名字）。
    *   **Public** (公开) 必须选中。
    *   点击 `Create repository`。
3.  **上传文件**：
    *   在仓库页面，点击 `uploading an existing file` 链接。
    *   将桌面上 **“待办”** 文件夹里的所有文件（`index.html`, `app-icon.png`, `manifest.json`）拖拽进去。
    *   等待文件上传完毕，点击底部的绿色按钮 `Commit changes`。

## 第二步：开启网页服务 (GitHub Pages)

1.  在仓库页面，点击顶部的 **Settings** (设置) 标签。
2.  在左侧侧边栏找到 **Pages** 选项。
3.  在 **Build and deployment** 下：
    *   **Source** 选择 `Deploy from a branch`。
    *   **Branch** 选择 `main` (或 `master`)，文件夹选择 `/(root)`。
    *   点击 `Save`。
4.  等待约 1-2 分钟，刷新页面，您会看到顶部出现一行字：
    > **Your site is live at...** (后面跟着一个链接)
    *   👉 **这个链接就是您的专属 APP 地址！**

## 第三步：安装到手机 (iPhone / Android)

### 🍎 iPhone (iOS)
1.  复制上面的链接，用 **Safari 浏览器** 打开。
2.  点击底部的 **分享按钮** (方框带向上箭头)。
3.  向下滑动，找到并点击 **“添加到主屏幕” (Add to Home Screen)**。
4.  点击右上角的 **“添加”**。
5.  🎉 现在桌面上有了一个名为“智效待办”的 APP 图标，点开即可全屏使用！

### 🤖 Android
1.  用 **Chrome 浏览器** 打开链接。
2.  点击右上角菜单 (三个点)。
3.  选择 **“安装应用”** 或 **“添加到主屏幕”**。
4.  确认添加。
